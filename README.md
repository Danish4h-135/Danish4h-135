**Problem:** Breach exposure is invisible until it's too late.

Ingests email breach data via HaveIBeenPwned API → routes through GPT-4 for automated risk scoring → generates OSINT-informed remediation plans. Built end-to-end in 36h at Vibeathon 2025 (Polaris × Replit).

### `[02]` SentinelAI

**Problem:** Linux host compromise is slow to detect without expensive tooling.

Multi-agent pipeline (ThreatTrace + VulnRank + PacketSpy) answers *"Is this machine compromised?"* in plain English within 30 seconds. Zero external telemetry. Isolation Forest anomaly detection on auth logs.

`Python` `Gemini Flash` `NVD API` `CVE/EPSS` `AST Analysis`

**→ [github.com/Cal2-0/Sentinel-IP](https://github.com/Cal2-0/Sentinel-IP)**

**Problem:** CI/CD pipelines are the new attack surface — and teams ship misconfigs daily.

Static scanner covering the full OWASP CI/CD Top 10: unpinned actions, exposed secrets, poisoned pipelines, excess permissions. Gemini-powered `--fix` flag auto-generates patched YAML.

`Python` `Gemini 1.5` `GitHub Actions` `OWASP CI/CD Top 10`
