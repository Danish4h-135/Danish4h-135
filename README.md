```
╔══════════════════════════════════════════════════════════════════════╗
  MISSION    : Real-time breach detection + AI-driven risk remediation
  STACK      : React · Vite · Node.js · OpenAI GPT · Google Gemini
               PostgreSQL · Drizzle ORM · HaveIBeenPwned API
  SECURITY   : AES-256 (all stored breach data) · bcrypt (auth)
  INFRA      : Replit Cloud · Persistent PostgreSQL storage
  LIVE URL   : darktrack.replit.app
  STATUS     : ██████████████████████  DEPLOYED & LIVE
```

**`HOW IT WORKS`** → Scans emails via HaveIBeenPwned API → Feeds breach data into OpenAI GPT for AI risk scoring + remediation advice → Google Gemini powers one-click **"Solve"** flows that walk users through 2FA setup & account hardening step-by-step → All breach records tracked per-user until explicitly resolved.
